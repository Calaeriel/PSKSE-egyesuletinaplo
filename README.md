Egyesületi napló, amelyben különböző kis érdekességek vannak a sportról és egy interaktív kalkulátor található benne. Regisztrációs füllel is rendelkezik, backend nélkül.
A JavaScript kódom a kalk.html-ben található. Egy olyan kalkulátor, amely kiszámítja a ló és ember súlyarányát, ezáltal megkapva, hogy lovagolhat-e az adott súlyú lovon az illető vagy sem. Továbbá, ha csak a testsúlyát írja, akkor kiszámolja, melyik kategóriájú lóra ülhet fel.

Használt betűtípus: Playfair Display
Betűtípus forrása: Google Fonts

<link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;700&display=swap" rel="stylesheet"> CSS-ben hivatkozva: font-family: 'Playfair Display', serif;
